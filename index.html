Lists

We have two types of lists. Ordered and unordered.

  1. Ordered List

    1. First item
    2. Second item
    3. Third item

    Reversed Ordered List

    This can be done by adding the reversed attribute to the ol tag.

    1. First
    2. Second
    3. Third

  2. Unordered List

    • First
    • Second
    • Third

  3. Description List

    4. This type of list is used to describe terms.

    USIU
    United States International University
    MFA
    Mpesa Foundation Academy

  4. Nested Lists

    5. This is a list within a list.

    1. Lower education
      1. Mlamba Pri School
      2. MFA
    2. Upper Education
      • Diploma Business information technology
      • BCs Applied computer technology

    6. This type of list is used to create a menu.
